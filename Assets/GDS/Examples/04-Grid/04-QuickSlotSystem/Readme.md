### Quick Slot System

This example demonstrates a way to implement a quick slot system as seen in games like Resident Evil.

A quick slot system is a way for the player to use important items quickly without opening their inventory.

When an item is dragged from the player’s inventory into a quick slot, it is copied to the new location, and the original item remains in its initial slot.

Pressing a slot key on the keyboard will “equip” the item. To simulate ammo consumption, press the **Shoot** button.

To remove an item from a quick slot, either click it or drag it out.

If an item that already exists in the quick slot menu is added again, the existing instance will be cleared.
